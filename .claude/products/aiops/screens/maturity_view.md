"우리 팀/내가 AI 를 얼마나 잘 쓰고 있는가?" 정량 측정. 개선 방향 제시.

## 4축 정의 (PA-010)

| 축 | 측정 |
|---|---|
| **Frequency** | 활용 빈도 (주간 호출 / 근무시간) |

| **Variety** | 사용 도구 다양성 (채널 수) |
| **Feedback** | 피드백 루프 (재질문·수정·좋아요) |

## 회귀 감지 로직 (PA-010-05)

```sql
SELECT u.id, u.name,
  curr.overall_score AS current_score,
  prev.overall_score AS prev_score,
  (curr.overall_score - prev.overall_score) / NULLIF(prev.overall_score, 0) * 100 AS change_pct
FROM users u
LEFT JOIN maturity_scores curr ON curr.user_id = u.id 
  AND curr.snapshot_week = (SELECT MAX(snapshot_week) FROM maturity_scores)
LEFT JOIN maturity_scores prev ON prev.user_id = u.id 
  AND prev.snapshot_week = (SELECT MAX(snapshot_week) FROM maturity_scores) - interval '1 week'
WHERE u.org_id = $1
  AND curr.overall_score - prev.overall_score < -prev.overall_score * 0.15;
```

회귀 감지 시 자동 `alerts INSERT (alert_type='maturity_regression')`.

```
Depth 점수: 82

세부 지표:
├ 평균 후속 질문 수: 3.8 회 (만점 4회 이상 = 100)
├ 최장 세션 깊이: 12 회 (상위 10%)
├ 즉시 종료 세션 비율: 23% (낮을수록 좋음)
└ "감사합니다" 등 의미없는 후속: 8%

개선 힌트:
• 세션 종료 후 재시작 대신 연속 깊이 유지
• 답변 내 후속 제안 질문에 응답해보기
```
