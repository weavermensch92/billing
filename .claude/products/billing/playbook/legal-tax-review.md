## 9-1. 법무 자문 (우선순위 최고)

### A. 리셀러 구조 (PB-001)

**확인할 것**:
- [ ] Gridge 가 "재판매자(resaler)"로 정의되어 AI 서비스를 다시 판매하는 구조의 법적 정합성
- [ ] 전자상거래법 적용 범위 (B2B 특성 반영)
- [ ] AI 벤더 약관 중 "기업 대리 관리 금지" 조항 전수 확인
  - Claude Team: Workspace 멤버십 기반 → **가능**
  - ChatGPT Team: 유사 → **가능**
  - Cursor Pro: 개인 계정 기반 → **조건부**
  - Lovable: 개인 계정 → **조건부**
  - OpenAI API: 기업 API 계약 → **가능**

### B. 결제·지급

**확인할 것**:
- [ ] **"결제 대행"** 용어 사용 금지 근거 (PG업 라이선스 필요)
- [ ] 가상카드(VCN) 이용이 **여신전문금융업법** 위반 여부
- [ ] 카드사와의 법인카드 계약 + 재판매가 **대리결제로 해석되지 않는지**

### C. 전자금융거래법

**확인할 것**:
- [ ] 현행 전자금융거래법상 PG 등록 의무 **비해당 근거** 서면화
- [ ] 2026.12.17 시행 개정안 (통신판매중개업자 부수 정산 대행) **영향 분석**
- [ ] 금감원 비규제 샌드박스 대상 여부

### D. 개인정보·보안

**확인할 것**:
- [ ] 고객사 멤버 정보 수집·보관의 **개인정보보호법** 적용
- [ ] 결제 정보 (VCN last4) 보관 범위 및 근거
- [ ] 감사 로그 3년 보관 근거
- [ ] 해지 후 30일 유예 + 완전 삭제의 적정성

### E. 계약서 핵심 조항

**확인할 것**:
- [ ] 서비스 이용 약관 (Alpha 템플릿 → 표준화)
- [ ] 크레딧백 프로그램 조건 명시 (매출 할인이지 환급 아님)
- [ ] 결제 사이클 티어 (월간·주간·선불) 조건 명시
- [ ] 해지·연체·분쟁 조항
- [ ] 전자서명·보관 요건

## 9-2. 세무 자문 (우선순위 최고)

### A. 부가세

**확인할 것**:
- [ ] Gridge 재판매가 부가세 **10% 과세 대상** 확정
- [ ] AI 서비스 원가 **매입세액 공제 여부**
  - 국내 매입 → 공제 가능
  - 해외 매입 (대리납부 등) → 특례 적용 확인
- [ ] 크레딧백이 **매출 감액 vs 할인** → 세금계산서 표기 방식
- [ ] 세금계산서 **공급가액 기준** (크레딧백 적용 후 금액으로 확정)

### B. 결제 사이클 티어별 세무 처리

**티어 2 (주간 중간 + 월간)**:
- [ ] 주간 내역서 = **선수금 처리** 가능 여부
- [ ] 월말 최종 세금계산서 1장 (공급시기: **월 말일**)
- [ ] 공급시기 해석 근거

**티어 3 (선불 예치금)**:
- [ ] 예치금 회계 처리 → **선수금** or **보증금**?
- [ ] 부가세 처리 시점 (예치 시 vs 차감 시)
- [ ] 예치금 반환 시 세무 처리

### C. 환율·환차

**확인할 것**:
- [ ] USD 원가 → KRW 환산 **환율 적용 시점** (매입 vs 매출)
- [ ] **환차손·환차익** 비용 인정 기준
- [ ] 고객 청구는 KRW 고정 → 환차는 Gridge 흡수 → 세무 영향 분석

### D. Anthropic 패스스루 회계 (PB-007)

**확인할 것**:
- [ ] 파트너 10% 할인의 **매입가 기준**
- [ ] 패스스루 매출 표시의 재무제표 처리
- [ ] 크레딧백 + 패스스루 **중첩 할인** 비용 인정
- [ ] 파트너십 할인 사항 고객 고지 의무?

### E. 연도 결산

**확인할 것**:
- [ ] 월말 **미청구 수익** 처리 (authorized·미settled)
- [ ] 매입 **미확정 건** (unsettled) 결산 일관성
- [ ] 해지 고객 D+30 데이터 삭제 후 세무 기록 **10년 보관** 방법
