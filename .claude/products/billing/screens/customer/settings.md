## 이메일 변경 재인증

이메일 변경 시:
1. 새 이메일에 인증 링크 발송
2. 24시간 내 클릭 확인
3. 확인 후 `members.email` 업데이트
4. `audit_logs INSERT (action='email_changed', visibility='both')`

## 권한

- **모든 멤버** 접근 (자기 영역만 수정)
- Owner / Admin 도 타 멤버 프로필 직접 수정 불가 — 초대 / 오프보딩만 가능
