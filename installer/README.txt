설치가 완료되었습니다. 다음 단계를 따라주세요.

 1. 현재 열려있는 Chrome / Edge 창을 모두 닫아주세요.

 2. Chrome / Edge 를 다시 실행하세요.
    → 자동으로 "Gridge Billing Fill Helper" 확장이 설치됩니다.

 3. 주소창에 chrome://extensions 또는 edge://extensions 입력해
    "Gridge Billing Fill Helper" 가 활성 상태인지 확인.

 4. 툴바의 Gridge 아이콘 우클릭 → "고정" 권장.

─── 사용 방법 ───────────────────────────────────────────────

 1. 먼저 https://app.gridge.ai 에 로그인 (SSO 지원)

 2. 카드 등록이 필요한 AI 서비스 결제 페이지로 이동:
      - https://console.anthropic.com/.../billing
      - https://platform.openai.com/.../billing
      - https://cursor.com/.../billing
      - 기타 지원 서비스

 3. 주소창 옆 Gridge 아이콘에 "$" 배지가 표시되면 클릭

 4. 팝업에서 사용할 VCN 선택 → "카드번호 복사"

 5. 결제 페이지의 카드번호 입력 필드에 Ctrl+V 로 붙여넣기

 6. 120초 후 클립보드가 자동으로 비워집니다 (보안).

─── 제거 ──────────────────────────────────────────────────────

 Windows 설정 → 앱 → "Gridge Billing Fill Helper" → 제거
 또는 제어판의 프로그램 제거 목록에서 삭제.

─── 주의사항 ──────────────────────────────────────────────────

 • 이 확장은 조직 정책으로 설치되어 Chrome/Edge 에
   "귀하의 조직에서 관리합니다" 배지가 표시될 수 있습니다.

 • Chrome/Edge 재시작 후에도 확장이 보이지 않으면 재부팅을 시도하세요.

 • VCN 카드번호는 Gridge 서버에서 일시 발급되며 브라우저 로컬에
   영구 저장되지 않습니다.
